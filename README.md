# 🏥 College Dispensary Management System

A full-stack web application to manage a college dispensary's daily operations — including user authentication, stock management, staff, and notifications.

---

## 🌐 Live Demo

- 🚀 **Frontend (React)**: [https://college-dispensary-pro1.onrender.com](https://college-dispensary-pro1.onrender.com)
- 🛠️ **Backend (Node + Express + MongoDB)**: [https://college-dispensary-pro.onrender.com](https://college-dispensary-pro.onrender.com)

---

## 🛠 Tech Stack

### Frontend
- React.js
- Axios
- React Router DOM


### Backend
- Node.js
- Express.js
- MongoDB with Mongoose
- JSON Web Token (JWT)
- CORS
- Nodemailer (for notifications/emails)

### Deployment
- Render (for both frontend and backend)
- GitHub (version control)
